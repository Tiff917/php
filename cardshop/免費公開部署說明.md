# 免費公開網址部署說明

## 目前狀況

`T's cashop` 是 `PHP + MySQL` 專案，不能直接部署到 GitHub Pages。

## 為什麼不能直接放 GitHub Pages

- GitHub Pages 適合靜態網站
- 這個專案需要 PHP 執行與 MySQL 資料庫
- 所以必須使用支援 `PHP + MySQL` 的主機

## 可用的免費方案

### InfinityFree

- 可提供免費子網域
- 支援 PHP 與 MySQL
- 適合這次期末專案做公開展示

## 我現在能先幫你完成的部分

- 整理好可上傳的網站檔案
- 整理好資料庫 SQL
- 整理好 GitHub 上傳內容
- 整理好部署用說明文件

## 最後卡住的地方

- 免費主機一定需要先註冊帳號
- 通常需要信箱驗證
- 這一步必須由你完成登入或提供已建立好的主機帳號

## 你完成帳號後我可以立刻接手做的事

1. 上傳 `card_shop` 專案
2. 建立 MySQL 資料庫
3. 匯入 `card_shop.sql`
4. 修改 `config.php` 連線資訊
5. 幫你確認公開網址可以開啟
